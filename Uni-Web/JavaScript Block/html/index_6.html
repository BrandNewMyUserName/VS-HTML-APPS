Завдання на лабораторну роботу

Створити Масив значень:
– ім’я друга;
– дата надходження;
– номер телефону (допускаються пусті значення).
За 3 дні потрібно відправити вітальну листівку, а в день народження поздоровити іменинника особисто по телефону. Вивести повідомлення про необхідність відправлення листівки чи телефонного дзвінка. Вивести додаткове повідомлення, якщо номер телефона не вказано.
